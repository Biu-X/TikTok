# 第六届字节青训营后端大项目-抖音

## 快速部署：

### 1. clone项目到本地

```bash
git clone https://github.com/hiifong/TikTok.git
cd TikTok
```

### 2. 修改`oss.endpoint`

修改`conf/tiktok.yml`的`oss.endpoint`为`运行电脑局域网ip:9000`

### 3. 运行项目

```bash
sudo docker compose up -d
```

#### 4. 修改抖声的`BaseURL`

双击“我”，在`baseURL`的输入框里填入`http://运行电脑局域网ip:8080`
